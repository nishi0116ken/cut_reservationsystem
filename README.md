# 機能一覧
### ユーザー側
●予約機能<br>
　・名前入力<br>
　・メニュー選択<br>
　・コメント機能<br>
　・instagram_id入力<br>
　・ご希望のへカラー<br>
　・ご希望のヘアスタイル<br>
●予約確認<br>
　・名前入力<br>
　・SNS_ID入力<br>

●管理者登録機能<br>
●ログイン機能,ログアウト機能<br>
●メニュー管理・スタッフ管理・予約管理<br>
  ・参照機能、追加機能、削除機能、修正機能<br>
